# Лабораторная работа №2 (вариант 5)

Учёт пациентов поликлиники — GUI на Python (PyQt6, MVC).

## Запуск

```bash
cd lab_2
source venv/bin/activate
pip install -r requirements.txt
python main.py
```

## Демо-данные

В `data/demo/` лежат три XML-файла (55, 60 и 52 записи). Загрузите любой через **Файл → Загрузить XML**.

Пересоздать файлы:

```bash
python scripts/generate_demo_data.py
```

Подробный отчёт и сценарии демонстрации — в [report.md](report.md).
